Core Advanced Excel Formulas

1. VLOOKUP

Formula: =VLOOKUP(lookup_value, table_array, col_index_num, [range_lookup])

Used to search a value in the first column of a table and return a value in the same row from another column.

2. INDEX + MATCH

Formula: =INDEX(return_range, MATCH(lookup_value, lookup_range, 0))

A more powerful alternative to VLOOKUP allowing left lookups and better flexibility.

3. IF & Nested IF

Formula: =IF(logical_test, value_if_true, value_if_false)

Use for logic-based conditions, decision trees, and validations.

4. SUMIFS / COUNTIFS

Formula: =SUMIFS(sum_range, criteria_range1, criteria1, ...)

Sum or count cells based on multiple conditions.

5. TEXT Functions

Useful ones:

  • LEFT(text, num_chars)
  • RIGHT(text, num_chars)
  • LEN(text)
  • CONCATENATE(text1, text2, ...)

Used to manipulate text values efficiently.

6. DATA VALIDATION

Restrict data entry using dropdown lists or custom formulas.

Example: Use =ISNUMBER(A1) to allow only numbers.

7. CONDITIONAL FORMATTING

Highlight cells based on values or rules.

Example: Color top 10% sales values or overdue dates automatically.
